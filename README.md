# WiFi DoS

A Python-based wireless network testing in Ethical Hacking and Cybersecurity

## Features

* Detect available wireless interfaces
* Enable monitor mode automatically
* Scan nearby wireless access points
* Display BSSID, channel, and ESSID information
* Continuous wireless monitoring
* CSV logging support

## Technologies Used

* Python 3
* Aircrack-ng
* Linux / Kali Linux

## Requirements

Before running the tool, install the required packages:

```bash
sudo apt update
sudo apt install aircrack-ng wireless-tools
```

Python version:

```bash
python3 --version
```

## Installation

Clone the repository:

```bash
git clone https://github.com/BharathkrishnaH4X/wifi_dos.git
```

Move into the project directory:

```bash
cd wifi_dos
```

Run the program:

```bash
sudo python3 wifi.py
```

## Supported Environment

* Kali Linux
* Linux systems with monitor mode capable adapters
