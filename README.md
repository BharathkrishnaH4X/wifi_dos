# WiFi DoS

A Python-based wireless network testing in Ethical Hacking and Cybersecurity

## Technologies Used

* Python 3
* Aircrack-ng - airmon-ng, airodump-ng, aireplay-ng
* Linux / Kali Linux

## Installation

Clone the repository:

```bash
git clone https://github.com/BharathkrishnaH4X/wifi_dos.git
```

Move into the project directory:

```bash
cd wifi_dos
```

Change the mode of File:
```bash
chmod +x *
```

Update the dependencies
```bash
./update.sh
```

Run the program:

```bash
sudo python3 wifi.py
```

Restart Network Manager:
```bash
sudo systemctl restart NetworkManager.service
```

## Supported Environment

* Kali Linux
* Linux systems with monitor mode capable adapters
